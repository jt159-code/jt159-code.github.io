我的第一篇博客

个人博客网址：Hexo

1、安装Node.js

https://nodejs.org/en/download/

2、安装Git

安装并进行环境变量设置

https://git-scm.com/download/win

3、安装hexo

npm install -g hexo-cli

4、初始化hexo，新建存储博客的文件夹

hexo init myblog

5、进入文件夹，安装一下npm

cd myblog npm install

6、启动服务站点

可以在本地访问了,访问http://localhost:4000/

hexo g hexo server

（一）在giee新建仓库

用户名+.gitee.io

8、安装一个hexo的上传插件deploy-git

npm install hexo-deployer-git --save

9、修改hexo配置文件指定仓库路径

找到Deployment加上（注意空格）

deploy: type: git repo: 仓库路径 branch: master

（二）Gitee Pages维护中，所以又去github上新建仓库实现个人博客

11、推送站点到github

hexo d

12、生成静态文件并推送文件

hexo clean && hexo g && hexo d

13、成功进入博客

Hexo

用Typora修改.md文件实现博客编写

15、主题

16、下载解压

18、配置文件_config.yml中修改为喜欢主题的名字

19、hero server启动即可访问
